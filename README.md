
# Parrot forwarder

2. Convert and categorize from multiple sources
3. Review 
4. (Optional) custom tagging

- MVVM + Repository
- Kotlin Flow
- Dagger Hilt (Dependency Injection)
- Retrofit (Networking)
- Navigation Component
- ktlint (Code Linting)
